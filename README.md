# 🔊 Звукопад

Аналог **Soundpad** на языке Rust — звуковая панель (soundboard) с графическим
интерфейсом на русском, поддержкой **своих горячих клавиш**, выводом звука
**одновременно на несколько устройств** (например, в виртуальный кабель и в
наушники), **глобальной кнопкой «Стоп»** и **имитацией нажатия микрофона** (PTT)
в играх во время воспроизведения.

## Возможности

- 🎹 **Любые горячие клавиши** — с модификаторами (Ctrl/Alt/Shift/Win) или
  вообще без них (например, просто `1` на Numpad). Работают глобально.
- 🔀 **Два устройства вывода одновременно:**
  - **Основное** — обычно виртуальный кабель (VB-Cable), чтобы слышали собеседники;
  - **Мониторинг** (наушники) — опционально, только чтобы вы слышали, что играет.
    Можно оставить «нет».
- 🎤 **Имитация микрофона (PTT)** — укажите клавишу голосового чата вашей игры,
  и Звукопад будет автоматически нажимать её во время воспроизведения звука,
  чтобы игра «думала», что вы говорите. После окончания — отпускает (с задержкой).
- ⏹ **Глобальная клавиша «Остановить всё»** — мгновенно стоп все звуки.
- 🔊 **Громкость каждого звука** + мастер-громкость + отдельная громкость мониторинга.
- ⏯ **Перезапуск или наложение** — повторное нажатие перезапускает звук (можно отключить).
- 📂 **Системный диалог** выбора файлов (mp3, wav, ogg, flac, m4a, aac).
- 💾 **Автосохранение** в `%APPDATA%/zvukopad/config.json`.
- 🪶 **Один `.exe`** в release-сборке.

## Требования для сборки

- **Rust** (тестировалось на 1.96).
- На Windows — линкер нативной ОС: **MinGW-w64** (`dlltool`/`gcc` в PATH)
  или **MSVC Build Tools**.

```bash
# GNU-тулчейн (по умолчанию):
cargo build --release

# или MSVC:
rustup default stable-x86_64-pc-windows-msvc
cargo build --release
```

Готовый файл: `target/release/zvukopad.exe`

## Подробная инструкция

### 1. Добавление звуков
Нажмите **➕ Добавить звук** справа → введите имя, нажмите **«Обзор…»** и выберите
звуковой файл. Готово — запись появится в центральном списке.

### 2. Назначение горячих клавиш
У каждой записи нажмите на кнопку клавиши и зажмите нужную комбинацию.
- **Можно использовать ЛЮБУЮ клавишу**, даже без модификаторов (например `Numpad1`).
- **Esc** — отменить назначение.
- **Backspace** — убрать клавишу.
- ✕ рядом с кнопкой — снять назначение.

### 3. Вывод звука «в микрофон» (как в Soundpad)

> 🌐 Скачайте и установите бесплатный **VB-Audio Virtual Cable**:
> **https://vb-audio.com/Cable/**

Затем в Звукопаде (раздел **«🔊 Устройство вывода»**):
- **Основное устройство:** выберите **«CABLE Input (VB-Audio Virtual Cable)»**.

В Discord / OBS / игре укажите микрофон = **«CABLE Output»**.
Теперь при нажатии горячей клавиши звук пойдёт собеседникам.

### 4. Мониторинг (слышать звук самому)
В разделе **«🎧 Устройство мониторинга»** выберите ваши наушники/колонки.

> ⚠️ **Это НЕОБЯЗАТЕЛЬНО.** Мониторинг нужен только вам, чтобы слышать, что играет.
> На передачу звука собеседникам он не влияет. Если не нужно — оставьте «— нет —».

У мониторинга есть своя громкость (отдельно от основной).

### 5. Имитация микрофона (PTT) для игр
В разделе **«🎤 Имитация микрофона»** укажите клавишу, которая в вашей игре
открывает голосовой чат (например `V` или `X`).

- Во время воспроизведения звука Звукопад **автоматически нажмёт** эту клавишу —
  игра решит, что вы говорите, и пустит звук.
- После окончания звука клавиша будет **отпущена** с заданной задержкой
  (настраивается, по умолчанию 300 мс — чтобы «хвост» точно ушёл).

> ⚠️ Имитация использует Windows `SendInput` со scan-кодами. Это работает в
> большинстве игр и Discord, но некоторые анти-читы могут блокировать
> синтетический ввод.

### 6. Глобальная кнопка «Стоп всё»
В разделе **«⏹ Глобальная клавиша»** назначьте клавишу — она мгновенно
остановит все играющие звуки из любого приложения.

## Структура проекта

```
src/
├── main.rs      — точка входа, окно eframe
├── app.rs       — интерфейс: список звуков, настройки, назначение клавиш, подсказка
├── audio.rs     — движок: два устройства вывода + PTT-интеграция (rodio)
├── ptt.rs       — имитация клавиши через WinAPI SendInput
├── hotkeys.rs   — глобальные горячие клавиши (global-hotkey), любая комбинация
└── config.rs    — конфигурация (JSON), автосохранение
```

## Используемые библиотеки

| Библиотека       | Назначение                          |
|------------------|-------------------------------------|
| `eframe`/`egui`  | Графический интерфейс (native)      |
| `rodio`          | Воспроизведение звука (cpal)        |
| `global-hotkey`  | Системные горячие клавиши           |
| `rfd`            | Нативный диалог выбора файла        |
| `serde`/`serde_json` | Сохранение конфигурации          |
| WinAPI `SendInput` | Имитация нажатия клавиши PTT      |
