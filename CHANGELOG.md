## [Unreleased]

### Добавлено
- Поддержка одновременного воспроизведения на два устройства (основное + мониторинг)
- Мгновенное применение изменений громкости без перезапуска звука
- Слайдер громкости мониторинга показывается только когда устройство выбрано
- GitHub Actions для автоматической сборки релизов под Windows
- GitLab CI для автоматической сборки релизов
- Подробная документация в README.md

### Изменено
- Рефакторинг аудио-движка: поддержка N устройств через HashMap
- Конфиг: заменены старые поля на `output_devices: Vec<OutputDeviceConfig>`
- UI: исправлены borrow-checker ошибки при выборе устройств

### Исправлено
- Громкость мониторинга не работала при выборе "нет" устройства
- Переключение устройств больше не останавливает воспроизведение

## [1.0.0] - 2024-07-01

### Добавлено
- 🎹 **Горячие клавиши** — любые комбинации с модификаторами или без (например, просто `Numpad1`)
- 🔀 **Два устройства вывода одновременно** — основное (VB-Cable) + мониторинг (наушники)
- 🎤 **Имитация микрофона (PTT)** — авто-нажатие клавиши голосового чата в играх через WinAPI SendInput
- ⏹ **Глобальная клавиша «Остановить всё»** — мгновенный стоп всех звуков
- 🔊 **Гибкая громкость** — мастер + на каждый звук отдельно + мониторинг
- ⏯ **Перезапуск или наложение** — повторное нажатие перезапускает звук
- 📂 **Системный диалог выбора файлов** — mp3, wav, ogg, flac, m4a, aac
- 💾 **Автосохранение** — конфиг в `%APPDATA%/zvukopad/config.json`
- 🪶 **Один `.exe`** — standalone release-сборка
- 🎨 **GUI на egui/eframe** — нативный интерфейс на русском языке

### Используемые библиотеки
- `eframe`/`egui` — GUI
- `rodio` — воспроизведение звука
- `global-hotkey` — системные хоткеи
- `rfd` — нативный файловый диалог
- `serde`/`serde_json` — конфигурация
- WinAPI `SendInput` — PTT имитация
